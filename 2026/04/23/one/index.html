关于hexo-blog使用

博客主题，静态博客

如果你在用 Hexo 搭建博客，一定听过 Fluid—— 它是目前最受欢迎、维护最活跃的 Material Design 风格主题之一，界面简洁大气、配置丰富、对中文极其友好，新手也能快速上手。
这篇文章就带你全面认识 Fluid 主题，看看它为什么值得选。
一、Fluid 主题是什么？
Fluid 是一款专为 Hexo 打造的 Material Design 风格开源博客主题，主打：
简洁优雅、现代清爽的界面
高度可定制
移动端 / PC 端自适应
功能齐全，开箱即用
持续更新、文档完善、社区活跃
项目主页：
https://github.com/fluid-dev/hexo-theme-fluid
官方文档：
https://hexo.fluid-dev.com/docs/guide/
二、Fluid 主题的核心亮点

  1. 高颜值 Material Design 设计
    卡片式布局、柔和阴影、清爽留白
    顶部 Banner 大图、毛玻璃导航栏
    文章排版舒适，阅读体验极佳
    支持自定义配色、背景、字体
  2. 完美支持暗色模式
    一键切换明 / 暗色
    可设置跟随系统自动切换
    代码高亮、图片、评论区同步适配夜间模式
  3. 丰富且实用的功能
    Fluid 几乎把博客需要的功能全部内置好了，不用自己瞎折腾：
    ✅ 文章目录（TOC）自动生成
    ✅ 代码高亮 + 代码复制按钮
    ✅ 图片灯箱放大查看
    ✅ 图片懒加载，打开速度飞快
    ✅ 文章字数统计 + 预计阅读时间
    ✅ 内置搜索功能
    ✅ 分类、标签、归档页面
    ✅ LaTeX 数学公式支持
    ✅ Mermaid 流程图支持
    ✅ 多种评论系统（Valine、Waline、Gitalk、Twikoo 等）
    ✅ 网站统计（不蒜子、LeanCloud、Umami 等）
    ✅ 页脚统计、备案信息、版权声明
  4. 高度自定义
    所有样式几乎都能改：
    导航栏、Banner、颜色、字体
    首页副标题打字机效果
    文章封面、头图、布局
    侧边栏目录位置、展开方式
    静态资源 CDN 可自由替换
  5. 对中文极其友好
    中文排版优化
    简体 / 繁体 / 英文多语言
    文档全中文，新手零障碍
    适配国内评论、统计服务
  6. 响应式，手机 / 平板完美适配
    移动端自动适配布局
    菜单转为简洁图标
    图片、文字自动缩放
    触摸操作流畅
    三、适合哪些人使用？
    Fluid 适合：
    想快速搭建好看又好用的博客
    不想写复杂前端代码
    需要数学公式、代码笔记、技术博客
    喜欢简洁优雅、可长期使用的主题
    希望主题持续更新、有文档、有社区
    可以说：
    如果你不知道选哪个 Hexo 主题，选 Fluid 准没错。
    四、快速安装（极简版）
    如果你想马上用：
    安装主题
    bash
    运行
    npm install –save hexo-theme-fluid
    在博客根目录新建 _config.fluid.yml，把官方配置复制进去即可。
    在 _config.yml 设置主题：
    yaml
    theme: fluid
    language: zh-CN
    创建关于页
    bash
    运行
    hexo new page about
    然后配置菜单、Banner、颜色，就能拥有一个和本站一样美观的专业博客。
    五、总结：为什么我推荐 Fluid？
    颜值高：现代简约，高级感拉满
    功能全：开箱即用，不用到处找插件
    配置简单：中文文档，改改配置就能个性化
    维护活跃：持续更新，bug 修复快
    生态成熟：教程多、问题容易解决
    性能好：懒加载、轻量、访问速度快
    如果你正在搭建 Hexo 博客，Fluid 绝对是最稳妥、最优雅、最省心的选择。
